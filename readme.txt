== Description ==

The plugin forms a flexible and functional personal account.

= Development and Translations =

If you're a developer and would like to contribute to the source code of the plugin you can do so via our [GitHub Repository](https://github.com/plechev-64/userspace).

Want to add a new language to UserSpace? Great! You can contribute via [translate.wordpress.org](https://translate.wordpress.org/projects/wp-plugins/userspace).

== Installation ==

1. Upload dir `userspace` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
